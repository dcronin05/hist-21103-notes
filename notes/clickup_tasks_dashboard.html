How to Setup clickup for HIST 21103

Step 1: Download the CSV File

Click the green "Download ClickUp CSV" button above. This exports the entire course timeline, points, descriptions, and subtask configurations into a cleanly formatted spreadsheet.

Step 2: Navigate to clickup Settings

In ClickUp, click your workspace profile avatar, go to Settings, select Imports/Exports, and click Start Import.

Step 3: Select Spreadsheet Import

Select the Excel or CSV option and upload the downloaded file (hist_21103_clickup_tasks.csv).

Step 4: Map the Column Fields

Map the spreadsheet columns to ClickUp standard fields as follows:

  • Task Name → **Task Name**
  • Description → **Description**
  • Start Date → **Start Date**
  • Due Date → **Due Date**
  • Priority → **Priority**
  • Points → **Custom Field (Number)**

Step 5: Alternative - Copy & Paste Method

If you prefer a manual setup, use the Task List tab. Click any "Copy" button to instantly save a task property to your clipboard and paste it directly into ClickUp. In ClickUp, you can paste the subtasks list as a single block to create separate subtasks automatically!
